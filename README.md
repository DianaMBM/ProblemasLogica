

# Problemas de Lógica en JavaScript y C#
Este repositorio contiene una colección de ejercicios de lógica resueltos utilizando los lenguajes de programación JavaScript y C#. Los problemas abordados son fundamentales para el desarrollo de habilidades en programación y son útiles para fortalecer la lógica algorítmica.

## Contenido del Repositorio
A continuación se describe el contenido de los diferentes archivos en este repositorio:

## Ejercicios en JavaScript
**Calcular IMC:** Un programa que calcula el Índice de Masa Corporal (IMC) basado en el peso y la altura proporcionados por el usuario.

**Calcular Potencia:** con Recursividad: Implementa una función para calcular la potencia de un número utilizando recursividad.

**Calcular Promedio:** Este ejercicio toma una lista de números y devuelve el promedio.

**Calcular Descuento:** Un programa que aplica un descuento a un precio inicial, basado en un porcentaje proporcionado por el usuario.

**Imprimir Tabla de Multiplicar:** Genera y muestra en pantalla la tabla de multiplicar de un número ingresado por el usuario.
**Menú de Operaciones:** Un menú interactivo que permite realizar operaciones matemáticas básicas como suma, resta, multiplicación y división.

**Factorial con Recursividad:** Calcula el factorial de un número utilizando una función recursiva.

## Ejercicios en C#
**Menú con C#:** Un programa que presenta un menú interactivo con varias opciones para que el usuario elija y realice diferentes operaciones.
